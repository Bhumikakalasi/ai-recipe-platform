# Whiskora - Smart Recipe Studio

**Full Stack AI Recipe Platform** built with **Next.js 15**, **Neon DB**, **Tailwind CSS**, **Shadcn/UI**, and more.

---

## ✨ Features

- AI-powered recipe generation (text + image)
- Smart recipe search & filters
- Meal planner & weekly calendar
- Personal recipe collection
- Ingredient substitution engine
- Beautiful responsive design with dark mode

## 🛠 Tech Stack

- **Frontend**: Next.js 15 (App Router) + TypeScript + Tailwind + shadcn/ui
- **Backend**: Next.js API Routes
- **Database**: Neon (PostgreSQL)
- **AI**: OpenAI / Grok API
- **Styling**: Tailwind CSS + Shadcn/UI

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/Bhumikakalasi/ai-recipe-platform.git
cd ai-recipe-platform

# Install dependencies
npm install

# Copy environment variables
cp .env.example .env.local

# Run development server
npm run dev
